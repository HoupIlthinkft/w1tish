# w1tish

#### Инструкция:
- Убедитесь что у вас установлен docker и docker-compose
1. Клонируйте репозиторий
`git clone https://github.com/KIriLOsck/w1tish.git`
2. Перейдите в каталог репозитория
`cd w1tish`
3. Запустите приложение
`docker-compose up`
4. Перейдите по ссылке http://localhost/

*Для остановки используйте* `docker-compose down`
